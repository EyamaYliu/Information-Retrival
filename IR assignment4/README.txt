This assignment requires installation of BeautifulSoup and urllib.






To use robot_base.py, input format should be:

	python3 robot_base.py <log_file route> <content_file route> <Web address to-be-crawled>

	Example:
		python3 robot_base.py mylog.log content.txt https://cs.jhu.edu/~yarowsky





To use lwp_parser.py, input format should be:

	python3 lwp_parser.py <Web address to-be-crawled>
	
	Example:
		python3 lwp_parser.py https://cs.jhu.edu/~yarowsky
